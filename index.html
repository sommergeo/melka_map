Geomorphological map of Melka Kunture (Ethiopia)

A geomorphological map of Melka Kunture to be published in Journal of Maps (under review). This repo contains the Web Map created with Leaflet and WMS provided by GeoServer and QGIS Server

Links

The map is available at

  • roceeh.net http://www.roceeh.net/ethiopia_geomorphological_map
  • gdi.geographie.uni-tuebingen.de http://gdi.geographie.uni-tuebingen.de/melka/geomorphology.html
  • Github test version https://sommergeo.github.io/melka_map/geomorphology.html
